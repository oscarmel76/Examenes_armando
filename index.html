Chapter 9
Area, Square Units & Perimeter

📦 What is Area? (pp. 127–130)

Pages 127–130

Definition

Area is the space inside a flat shape. Measured in square units.

How to Count Squares

  • Count unit squares inside the shape.
  • Each small square = 1 square unit.
  • Two half-squares = 1 whole.
Area = Length x Width
A = L x W

✏️ Try It!

  1. Rectangle 6x4 units → Area=?
    → 24 sq units
  2. 5 rows of 8 squares → Area=?
    → 40 sq units
  3. Square side=7 cm → Area=?
    → 49 cm²

📏 Square Units: cm², in², m², ft² (pp. 139–154)

Pages 139–154

Which unit?

  • cm² / in² → small: book, card, photo
  • m² / ft² → large: room, garden, field
💡 Tip: Area always uses ² because you measure in TWO directions.

✏️ Try It!

  1. Square side=9 cm → A=?
    → 81 cm²
  2. Room 6m x 5m → A=?
    → 30 m²
  3. Photo 8in x 6in → A=?
    → 48 in²

📐 Perimeter AND Area (pp. 167–172)

Pages 167–172

⚡ The Big Difference

  • Perimeter = distance AROUND the shape → ADD all sides → simple units
  • Area = space INSIDE → MULTIPLY → square units (²)
🖼️ Perimeter vs Area
🟦 Rectangle
🟩 Square
🔺 Triangle

✏️ Try It!

  1. Rectangle L=12cm, W=5cm → P=? A=?
    → P=34cm | A=60cm²
  2. Square side=10ft → P=? A=?
    → P=40ft | A=100ft²
  3. Triangle sides: 7,8,9m → P=?
    → P=24m

🔷 Compound Shapes (pp. 167–172)

Key Math in Focus Skill!

What is a Compound Shape?

Two or more rectangles joined together (L-shape, T-shape, staircase).

🪜 Steps

  1. Draw a line to split into rectangles.
  2. Find the area of each part.
  3. Add all parts together = Total Area!
🔷 L-Shape Example 1
Part A (top)8 cm x 4 cm = 32 cm²
Part B (bottom)5 cm x 5 cm = 25 cm²
Total = 32 + 25 = 57 cm² ✅
🔷 L-Shape Example 2 (wider)
Part A (top)10 cm x 3 cm = 30 cm²
Part B (right)5 cm x 6 cm = 30 cm²
Total = 30 + 30 = 60 cm² ✅
🔷 T-Shape Example
Part A (top bar)11 cm x 3 cm = 33 cm²
Part B (stem)4 cm x 5 cm = 20 cm²
Total = 33 + 20 = 53 cm² ✅
🔷 Staircase Shape (3 parts!)
Total = 16 + 32 + 48 = 96 cm² ✅ (3 parts, same method!)

✏️ Practice

  1. L-shape: A=6x2=12cm², B=3x4=12cm². Total=?
    → 24 cm²
  2. T-shape: A=10x2=20m², B=3x5=15m². Total=?
    → 35 m²
  3. L-shape: A=8x4=32ft², B=3x6=18ft². Total=?
    → 50 ft²
  4. A compound shape has total 80cm². Part A=45cm². Part B=?
    → 80-45=35 cm²

🎯 Chapter 9 Review (pp. 195–200)

📋 Formula Cheat Sheet

  • Area rectangle: A = L x W
  • Area square: A = s x s = s²
  • Compound shape: Split → each area → add together
  • Perimeter rectangle: P = 2L + 2W
  • Perimeter square: P = 4 x s
  • Perimeter triangle: P = a + b + c
  • Small → cm²/in² | Large → m²/ft²
  • Area uses ² | Perimeter uses simple units

⏰ Elapsed Time (pp. 247–250)

Jump Strategy 🦘

  1. Find start and end time.
  2. Jump by whole hours first.
  3. Count remaining minutes.
  4. Add hours + minutes = elapsed time ✅
💡 Example: Start 2:15 PM → End 4:45 PM
2:15→4:15=2hrs · 4:15→4:45=30min → 2hrs 30min

✏️ Try It!

  1. Start 1:00PM · End 3:30PM
    → 2hrs 30min
  2. Start 9:15AM · End 11:45AM
    → 2hrs 30min
  3. Start 8:20AM · End 11:50AM
    → 3hrs 30min

🎯 Chapter 10 Review (pp. 255–258)

📋 Cheat Sheet

  • Short hand = hours · Long hand = minutes
  • Each number = 5 minutes
  • :15 = quarter past · :30 = half past · :45 = quarter to
  • 1 hr = 60 min · 1 min = 60 sec
  • Hours→min: x60 | Min→hours: ÷60
  • Elapsed time → Jump Strategy
  • AM = morning | PM = afternoon/night
